Dancing Game

Inspiration

I’m a dancer. I like dancing very much. I wanted to create a dance game that we can play anytime and anywhere. This project is a replica of a famous dance game - just dance. I hope I can complete this project with what I have learned.

Project Overview

Summary: As a passionate dancer, I wanted to create a dance game that anyone can play anytime and anywhere. This project replicates the popular dance game Just Dance using innovative technology to enhance interactivity and performance tracking.

Objective: To build a fun, immersive dance game that utilizes motion recognition to track and evaluate players’ movements in real time.

Problem Statement or Inspiration

Dance is a universal language of self-expression, yet structured dance activities often require specific environments or equipment. Inspired by Just Dance, I aimed to make dance more accessible and interactive by combining technology and creativity using p5.js.

Features and Technology

  • Video Integration: The game includes videos from Just Dance to provide dance sequences for players to follow.
  • Motion Recognition: Leveraging p5.js with motion-tracking libraries (e.g., PoseNet), the game detects and evaluates player movements using a webcam. Skeleton-based tracking compares the player’s motions to predefined dance prompts, ensuring accurate feedback.
  • Immersive Gameplay: Real-time movement analysis and visual feedback make the experience engaging and educational.

Tools and Methodology

  • Tools Used: p5.js for motion recognition, interactive visuals, and game logic; PoseNet (via p5.js) for skeleton tracking; HTML/CSS for creating a user-friendly interface; and video editing tools to prepare dance videos.
  • Development Process: Used PoseNet to extract keypoints for skeleton tracking in p5.js, built an interactive interface that overlays real-time feedback on top of the dance video, and designed algorithms to compare player movements with reference movements in the video.

Outcomes and Achievements

  • Working Prototype: Built a fully functional game that tracks player movements using the webcam and provides real-time feedback.
  • User-Friendly Design: Created an accessible interface that encourages players of all skill levels to participate.
  • Integrated Learning: Applied concepts of motion recognition, human-computer interaction, and game design in a cohesive project.

Challenges and Learnings

  • Challenges Faced: Fine-tuning PoseNet to ensure accurate tracking of complex dance movements and balancing game performance with browser compatibility.
  • What I Learned: Gained expertise in using p5.js for interactive applications and developed a deeper understanding of how to merge art, technology, and user experience design.

Future Directions

  • Expand the library of dance routines with different genres and difficulty levels.
  • Incorporate multiplayer modes for competitive or collaborative gameplay.
  • Optimize for mobile platforms, enabling users to play the game on their phones.
