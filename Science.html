Escape Room Horror Game

The game is an interactive, web-based escape room built using p5.js. Players navigate through challenges and puzzles by clicking on various elements to solve them, progressing through different levels. The game includes unique graphics and sound effects to enhance the immersive experience. It requires logical thinking and observation, combining visuals, audio, and interactive mechanics to create an engaging gameplay experience.

Dancing Game

The game is an interactive, web-based dancing experience developed using p5.js. Players engage by following dance movements or patterns on the screen and interacting through clicks or keystrokes. The goal is to mimic or follow the rhythm, with each successful action contributing to a complete dance routine. The game features visually engaging graphics and synchronized music, creating an immersive and fun dance simulation. The mechanics challenge players' timing and rhythm skills, combining visual cues and audio elements for an enjoyable dance game experience.
